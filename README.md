
# Next.js Frontend Project

This project is set up with Next.js, TypeScript, and CSS/SCSS modules. No CSS frameworks are included.


## Getting Started

1. Install dependencies:
	```sh
	npm install
	```
2. Run the development server:
	```sh
	npm run dev
	```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Styling
- Use `.module.scss` files for component-level styles.
- No CSS frameworks (e.g., Bootstrap, Tailwind) are included or allowed.

## Project Structure
- `pages/` or `app/`: Main application routes
- `styles/`: Global and module SCSS files
